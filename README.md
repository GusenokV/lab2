## Варіант завдання

Розрахунок параметрів виконується на основі номера залікової книжки (№ = 5112):

1. **C3** (Тип даних): 5112 ÷ 3 = 1704, остача **0**
   * Тип змінних: `StringBuilder`.

2. **C17** (Дія з текстом): 5112 ÷ 17 = 300, остача **12**
   * Завдання: Відсортувати слова заданого тексту за кількістю входжень визначеного символу в них.

---

## Опис реалізації

Програма складається з двох класів:

* **`TextProcessor`** — клас з логікою обробки тексту. Всі операції виконуються виключно через `StringBuilder`; перетворення до `String` дозволяється лише при введенні та виведенні даних.
* **`Main`** — виконавчий клас, що зчитує текст і символ з клавіатури та виводить результат.

Програма виконує наступні кроки:

1. Користувач вводить текст з клавіатури (завершення — порожній рядок) та символ для сортування.
2. Метод `extractWords()` розбиває текст на слова, відкидаючи пробіли та розділові знаки.
3. Для кожного слова метод `countOccurrences()` підраховує кількість входжень заданого символу.
4. Метод `sortWordsByCharOccurrence()` сортує слова за зростанням цього значення алгоритмом insertion sort (стабільне сортування).
5. Результат виводиться у консоль.
